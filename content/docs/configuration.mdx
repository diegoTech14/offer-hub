OFFER-HUB is configured through environment variables in `backend/.env`. This page documents every available option.

## Required Variables

These must be set for the application to start:

```env
# Supabase connection
SUPABASE_URL=https://your-project.supabase.co
SUPABASE_ANON_KEY=your-anon-key
SUPABASE_SERVICE_ROLE_KEY=your-service-role-key

# Auth
JWT_SECRET=your-secret-key-min-32-chars

# Server
PORT=4000
NODE_ENV=development
```

## Stellar / Blockchain

```env
# Stellar network
STELLAR_NETWORK=testnet
STELLAR_RPC_URL=https://soroban-testnet.stellar.org
STELLAR_HORIZON_URL=https://horizon-testnet.stellar.org
```

<Callout variant="note">
  Use `mainnet` for production. The Soroban RPC endpoint changes between networks.
</Callout>

## OAuth Providers

All OAuth variables are optional. Only configure the providers you want to support:

```env
# Google
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=

# GitHub
GITHUB_CLIENT_ID=
GITHUB_CLIENT_SECRET=

# Apple
APPLE_CLIENT_ID=
APPLE_TEAM_ID=
APPLE_KEY_ID=
APPLE_PRIVATE_KEY=
```

## Email

```env
# SMTP (for transactional emails)
SMTP_HOST=smtp.example.com
SMTP_PORT=587
SMTP_USER=noreply@example.com
SMTP_PASS=your-smtp-password
```

## Frontend Variables

The frontend uses Next.js environment variables (prefix with `NEXT_PUBLIC_` to expose to the browser):

```env
NEXT_PUBLIC_API_URL=http://localhost:4000
NEXT_PUBLIC_STELLAR_NETWORK=testnet
```

## Variable Reference Table

| Variable | Required | Default | Description |
|----------|----------|---------|-------------|
| `SUPABASE_URL` | Yes | — | Your Supabase project URL |
| `SUPABASE_ANON_KEY` | Yes | — | Supabase anon (public) key |
| `SUPABASE_SERVICE_ROLE_KEY` | Yes | — | Supabase service role key |
| `JWT_SECRET` | Yes | — | Secret for signing JWTs |
| `PORT` | No | `4000` | HTTP server port |
| `NODE_ENV` | No | `development` | Runtime environment |
| `STELLAR_NETWORK` | No | `testnet` | `testnet` or `mainnet` |

<Callout variant="danger">
  Never commit your `.env` file to version control. It is already listed in `.gitignore`.
</Callout>
